## рҹҡҖ Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** v14 or higher вҶ’ [Download](https://nodejs.org/)
- **npm** or **yarn**

### вҡҷпёҸ Installation

```bash
# 1. Clone the repository
git clone https://github.com/hjanoti/my-portfolio.git

# 2. Navigate into the project
cd my-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
```

рҹҢҗ Open [http://localhost:3000](http://localhost:3000) in your browser.

### рҹҸ—пёҸ Production Build

```bash
npm run build
```

Output will be in the `build/` folder, ready for deployment.

## рҹҺЁ Customization Guide

### рҹ‘Ө Personal Info
- Update name, tagline, and intro text in `src/components/Hero.js`
- Replace profile picture at `public/assets/profile.png`

### рҹ–јпёҸ Projects
- Edit project cards in `src/pages/Projects.js`
- Add screenshots to `public/assets/projects/`
- Update GitHub repo links and live demo URLs

### рҹӣ пёҸ Skills
- Add or remove tech skills in `src/components/Skills.js`
- Use Tailwind classes to adjust icon colors and layout

### рҹ“¬ Contact
- Update email, phone, and social links in `src/pages/Contact.js`
